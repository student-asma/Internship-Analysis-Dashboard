# Internship Program Analysis Dashboard

## Project Overview
This project was developed as the first task of my Data Analytics Internship. The dashboard analyzes internship enrollment, completion rates, dropout trends, mentor interaction, and internship duration using Power BI.

## Features
- KPI Cards
- Department-wise Completion Analysis
- Internship Status Distribution
- Internship Duration Analysis
- Mentor Interaction Analysis
- Interactive Filters
- Key Insights

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Visualization
